# Uslugi

Android-приложение для девушек и женщин с подбором услуг: маникюр, педикюр, цветы, косметика и другое. Стартовая версия на Java включает карту города и список услуг с рейтингами и отзывами.

## Что дальше
- Подключить реальные данные по городам и категориям.
- Добавить фильтры по цене и времени.
- Настроить авторизацию и сохранение отзывов.
