# Uslugi

Android-приложение для девушек и женщин с подбором услуг и магазинов рядом. Версия на Java реализует MVVM, Navigation Component и мок-данные для экранов: главный, карта, карточка услуги, запись и профиль.

## Что уже есть
- Поиск по названию и категории.
- Карта с ближайшими точками услуг и магазинов.
- Рейтинги и отзывы с возможностью оставить отзыв.
- Запись на услугу с выбором даты и времени.
- Профиль с будущими и прошедшими записями.
